# Towny 插件使用指南

 

Towny 是一个土地保护和 城镇/国家 模拟插件，允许玩家创建、保护和管理城镇，拥有土地，并组建国家和联盟。本指南旨在全面描述 Towny 插件的各项功能。指南的第一部分（"**城镇基础**"）作为 Towny 的入门介绍，描述了创建城镇的初步步骤、如何购买土地以及各种基本功能。土地保护及其配置方法在第二部分（"**权限**"）中描述。其他部分与 Towny 插件的其他所有功能相关，涉及城镇、国家和插件提供的设置。

 

## 基本概念

要理解 Towny 的工作原理，需要知道“区块与城镇天”、“城镇”、“居民”、“银行与税务”和“国家”的概念。

**区块**是世界的 16x16 部分，在垂直方向上无限延伸。要查看世界上的区块分隔，可以使用 F3+G 来显示区块边界。世界中的所有区块默认都是“荒野（<u>Wildness</u>)”，在荒野中服务器上的所有玩家都可以建造、破坏、访问门、箱子等。荒野中的 PvP 是启用的。

**城镇日**（<u>Towny Day</u>）即为现实中的一天。Towny按城镇日工作。当每个城镇日开始时会在聊天栏中显示。

 

**城镇**是一群玩家（称为居民，至少 <u>1</u> 人）在某个城镇名称下的集合，城镇可以将荒野中的区块圈为己用以保护其免受其他玩家侵害。城镇从荒野中夺取的区块称为领地或城镇地皮（<u>Plot</u>）。城镇地皮可以不被任何居民拥有——这些称为无主城镇地皮，或被居民拥有——这些称为居民地皮。

 

**居民**是指从属于任意城镇的玩家（<u>Resident</u>）。首个创建城镇的玩家称为市长（<u>Mayor</u>）。每个城镇都有一个市长，其是城镇的所有者，可以访问自己城镇内的所有城镇地皮以及所有相关命令。市长还可以通过授予其他居民城镇等级来让他们帮助处理城镇事务。

 

城镇**银行**是城镇的资金虚拟存储库；只有市长和副市长可以从城镇银行取款，但所有居民都可以向其中存款。市长和副市长还可以通过输入`/t bankhistory`查看其城镇银行当前城镇日的银行交易历史。

城镇的正常运作需要“**交税**”。服务器会在每个城镇日开始时向所有城镇的银行收取费用来允许城镇正常运作。在本服务器中，若城镇的银行余额不足以维持存在，城镇会进入“**破产**”状态，同时银行开始负债。进入破产状态后，城镇的部分功能将受到限制。任何居民都可以在此期间向银行存钱来解除破产状态。城镇最多可以在破产状态下维持 <u>30</u> 天。超过期限或负债达到最高限制，城镇将被删除。

除服务器要求的税费外，加入国家的城镇需要向从属国家交国家税；拥有城镇地皮的居民需要向城镇市长交地皮税；任何居民都需要向城镇交城镇税。

无法支付国家税的城镇会被移出国家；无法支付地皮税的居民会失去其地皮；无法支付城镇税的居民（**除助手**）会被移出城镇。

 

**国家**是一群城镇（至少 <u>1</u> 个）的集合，国家中所有城镇的居民都是其国家成员。

 

## 城镇创建与玩家管理

要创建您的城镇，使用 `/t new (您的城镇名称)` 或 `/t create (您的城镇名称)`。此操作会从您的余额中扣除资金。城镇名称只接受<u>英文、数字和英文下划线</u>。

 

创建城镇后，您也会被任命为城镇的市长。您可以**随时**使用 `/t set name (新名称)` 更改城镇名称。

 

使用 `/t add (玩家名称)`邀请另一名玩家加入您的城镇。他们可以输入`/accept (您的城镇名称)`加入。要取消对玩家的邀请，请用 `/t invite -(玩家名称)`。

 

**任何**居民（<u>市长除外</u>）都可以通过输入 `/t leave` 离开城镇。作为市长，您还可以使用  `/t kick (玩家名称) `  将居民从城镇中踢出。

 

城镇**不允许**城镇外的**任何人**访问其领地，<u>只有</u>您自己城镇的居民可以。如果您也**不希望**<u>居民</u>随意使用城镇，现在可以通过简单地执行 `/t set perm off` 来**阻止**他们访问。 <u>您的居民和您</u>  可以购买自己的土地来使用，与公共城镇空间和其他人**分开**！

 

###  领地占领

 

占领（<u>Claim</u>）是从荒野中<u>夺取一个区块</u>并将其**添加**到<u>您的城镇中</u>，以扩展和保护您土地的过程。您的城镇占据的所有区块都称为领地或城镇地皮。在创建城镇时，您占领了您所在的区块。

 

要占领一个区块，请在与您现有领地相邻的荒野区块上站立时输入 `/t claim`。此操作会从<u>城镇银行</u>中**扣除资金**。

 

要占领一个与您现有领地不相邻的区块，输入 `/t claim outpost`。此操作从<u>城镇银行</u>中扣除的资金比占领相邻区块**要多**。

 

然后您可以照常输入 `/t claim` 来占领相邻区块。每个分离的领地组称为一个“**城镇前哨**“。城镇前哨中占领的第一个区块称为“**城镇前哨出生区块**“，而您在被占领区块中站立的位置称为“**城镇前哨出生点**”。

 

要从您的城镇中移除/放弃（或"**抛弃**"）您所在的区块，使用 `/t unclaim`。放弃城镇前哨的最后一个区块也会删除该城镇前哨。

 

城镇在任何时候可能拥有的领地数量是有限制的。该限制由<u>城镇规模决定</u>。

 

 城镇规模

 

城镇规模是城镇拥有的领地数量与其可能拥有的最大领地数量之比。您可以在城镇信息屏幕（”/t”）中查看城镇规模。您的城镇可以占领的可用区块数量可以通过三种领地奖励来增加。

 

失去城镇居民和失去国家成员意味着您也会失去城镇规模中相应的领地奖励。

 

城镇规模和领地奖励的确切值列在本指南的最底部。

 

###  城镇维护费用

 

城镇维护费是服务器在每个 **Towny日** <u>开始时</u>自动从您的城镇银行中扣除的金额，以便城镇能够继续存在。城镇银行中没有足够的资金支付维护费意味着您的城镇将**陷落**。城镇地皮<u>越多</u>，每天支付的城镇**维护费就<u>越高</u>**。

 

在 Towny 中，居民可以随时使用 `/t deposit (金额)` 手动将资金从他们的余额转移到城镇银行。

:::tip

在国家中，上述命令只能在自己城镇的主区块或银行地块上使用。

 :::

通常，在<u>规划城镇</u>时考虑城镇维护费用**很重要**。一个非常大且有不必要领地的城镇将比一个规模较小且组织良好的城镇不必要地昂贵。

 

###  购买和拥有地皮

 

您城镇的每个居民(包括市长)都可以拥有城镇地皮，他们可以在那里拥有自己的建造和使用空间，空间大小<u>取决于</u>市长可以或想要给予的大小，其他居民通常无法访问。居民拥有的城镇地皮称为他们的居民地皮。居民完全有权访问自己的居民地皮以及所有相关命令。居民可以**拥有多个和分离**的居民地皮。

 

居民**必须**先购买城镇地皮，才能使其成为自己的居民地皮。

 

**市长**要撤回地皮的销售，可以输入 `/plot notforsale`。

 

居民可以在其拥有的居民地皮之一上站立时使用 `/plot unclaim`，以**移除**其对它的所有权（"**放弃**"其居民地皮）。

 

居民拥有自己的土地是任何城镇的必要功能，因为居民用于建造和保护物品的土地与公共城镇土地区分开来。组织良好的城镇通常允许每个居民在城镇周围的便利位置拥有居民地皮，并且所有居民地皮都靠近或彼此相邻。为了更容易的城镇管理，强烈建议这样做！

 

###  城镇出生点

 

每个城镇都有一个主区块，其存在是因为城镇必须始终<u>至少</u>拥有一个被占领的区块。创建城镇时，您所在的区块成为第一个主区块，您的确切位置成为第一个城镇出生点。城镇出生点位于主区块内，并由地面上的粒子指示。

 

主区块和城镇出生点的位置可以由市长更改。要更改主区块的位置以及使用 `/t spawn`时人们传送到城镇的位置，您需要先执行 `/t set homeblock`，然后执行 `/t set spawn`。您的确切位置将是新的城镇出生点。

 

主区块等同于"**城镇中心**"的概念。城镇出生点和主区块的位置最适合在城镇的中央位置，居民可以快速访问城镇的其余部分。

 

###  加入国家

 

成为国家的一部分对城镇有许多好处。加入国家的城镇可以获得更多的可占领区块，并获得与经济、影响力、沟通和社区相关的各种其他游戏玩法益处。加入国家的城镇可能还必须从其城镇银行支付每日的城镇国家税收——除了维护费之外。您的城镇每天支付的城镇国家税收由**国家领导人**设定，可以通过输入 `towny prices`（在底部）查看。

通常，您的城镇可以在被<u>邀请</u>后加入国家。市长可以通过输入 `/accept (邀请国家的名称)”`接受国家邀请，或通过输入`/deny (邀请国家的名称)`**拒绝**邀请。

 ::tip

另一方面，如果您不想加入其他国家，您也可以创建自己的国家！创建和管理国家将在本指南后面介绍(该部分待完善)

 :::

每个领地都有权限，它们<u>允许或拒绝</u>某些玩家在其中执行某些操作。在荒野区块中，所有权限对服务器上的每个玩家都启用，这无法更改。权限是 Towny  作为插件能够在城镇内保护您的建筑、箱子等的原因。正确配置城镇地皮的权限很重要，以便只有您希望访问城镇地皮的人才能拥有这些权限。

您可以在城镇信息屏幕上查看城镇的权限设置。您可以在地皮信息屏幕（`/plot info`）上查看单个领地的权限设置。

在查看权限时，如果某个组被拒绝该权限类型，它将显示为破折号 **“-”**。如果该组被允许该权限类型，它将显示其名称的首字母。这些组称为**“权限等级“**。

 

####  配置权限

 

下面解释权限类型、权限等级和权限类别。

 

#####  权限类型

 

4 种权限类型中的每一种都与不同的操作相关。

 

######  权限等级

 

权限等级类似于组的概念，并且它们具有层次结构。无主城镇地皮与居民地皮（拥有的城镇地皮）具有不同的权限等级。

城镇权限等级适用于无主城镇地皮：
- 市长 (M)
- 副市长 (A)
- 助理 (s)
- 居民 (r)
- 盟国 (a)
- 外来者 (o)

 

地皮权限等级适用于居民地皮：
- 地皮所有者 (f)
- 市长 (M)
- 副市长 (A)
- 助理 (s)
- 居民 (r)
- 盟国 (a)
- 外来者 (o)

 

######  权限类别

 

有 **4 类**权限需要配置。每个类别适用于某些城镇地皮。

 一般城镇权限和单个城镇权限适用于无主城镇地皮。 

居民权限和地皮权限适用于居民地皮。当居民购买城镇地皮时，城镇权限变成它的居民/地皮权限；当居民的地皮所有权被移除时，居民/地皮权限再次变成它的城镇权限。 
- 为城镇中所有无主城镇地皮配置一般城镇权限，使用 `/t set perm (城镇权限等级) (权限类型) (on/off)`
- 为您所在的无主城镇地皮配置单个城镇权限，使用 `/plot set perm (城镇权限等级) (权限类型) (on/off)`
- 为您拥有的所有居民地皮配置居民权限，使用 `/res set perm (地皮权限等级) (权限类型) (on/off)`
- 为您所在的居民地皮配置地皮权限，使用 `/plot set perm (地皮权限等级) (权限类型) (on/off)`

 

 

###  信任 (Trust)

 

信任允许您为城镇中的任何领地向服务器上的任何玩家授予所有权限类型（建造、破坏、切换、物品使用）。

- 城镇信任：要使用城镇的所有领地信任玩家，使用`/t trust add (玩家名称)`。要从**城镇**撤销玩家的信任，使用 `/t trust remove (玩家名称)`。

- 地皮信任：要信任您所在的城镇地皮的玩家，使用 “/plot trust add (玩家名称)”。要从地皮撤销玩家的信任，使用 “/plot trust remove (玩家名称)”。

将玩家从受信任者中移除并**不一定**意味着他们停止访问城镇或城镇地皮。这意味着您为城镇地皮配置的权限将再次适用于他们。

###  玩家权限

 

您还可以允许服务器中的特定玩家在城镇地皮中拥有某些权限，或拒绝特定玩家在城镇地皮中拥有某些权限。城镇地皮可以是无主城镇地皮或居民地皮。

 

要将玩家添加到此配置，输入 `/plot perm add (玩家名称)`。要从此配置中移除玩家，输入 `/plot perm remove (玩家名称)`。

 

添加玩家后，您可以使用 `/plot perm gui` 并单击菜单中的玩家头颅来配置他们的权限。菜单允许您通过单击菜单**中间的 4 个**按钮为玩家单独配置每种权限类型。

编辑权限后，您可以单击左下角的保存按钮以保存配置。改为单击右下角的删除按钮会将玩家从其在地块中的每个玩家权限中移除。

 

从每个玩家权限中移除玩家并不一定拒绝他们在城镇地皮中的所有权限类型。这意味着您为城镇地皮配置的权限将再次适用于他们。

 

###  重置权限

 

您可以使用 `/t set perm reset` 将当前的一般城镇权限应用于**城镇中每个无主城镇地皮**。此操作撤销所有无主城镇地皮上单个城镇权限的权限覆盖，以防它们在占领后从其默认的一般城镇权限更改。

 

您可以使用 `/res set perm reset`将您当前的居民权限设置应用于<u>您拥有</u>的**每个居民地皮**。此操作撤销您任何居民地皮上地皮权限的权限覆盖，以防它们在购买后从其默认居民权限更改。

 

您可以使用 `/plot set perm reset` 将您当前的居民权限设置（<u>如果地皮由居民或您拥有</u>）或当前城镇权限设置（<u>如果城镇地皮是无主的</u>）应用于您所在的地皮。此操作：

1.  对于无主城镇地皮，撤销单个城镇权限的权限覆盖，以防它们在占领后从其默认的一般城镇权限更改，并在其上应用一般城镇权限；
2.  对于居民地皮，撤销手动地皮权限的权限覆盖，以防它们在购买后从其默认居民权限更改，并在其上应用您的居民权限。

 

重置权限不会移除受信任的玩家，也不会重置每个玩家的权限设置。

 

 

###  城镇信息

 

使用 `/t (城镇名称)` 查看城镇的城镇信息屏幕。您**自己的**城镇的城镇信息屏幕可以通过输入 `/t`查看。

 

城镇信息屏幕：

- 城镇名称

- 领地数量

- 领地奖励

- 最大领地数

- 居民数量（在线/总数）

- 市长

- 城镇等级

- 城镇类型（开放/关闭，公开/私人）

- 城镇银行余额

- 城镇税收类型和金额

- 城镇维护费用

在国家中，城镇信息屏幕还将显示：

- 国家名称

- 国家等级

 

###  城镇等级

 

在城镇中，居民可以拥有等级以指示他们在城镇中的相对位置。每个等级拥有不同的权力。 

当居民加入城镇时，他们没有城镇等级。城镇的所有居民都可以向城镇银行存款，拥有城镇地皮，更改自己的居民地皮的地皮权限，并重命名自己的居民地皮。

市长和副市长有能力使用 `/t rank (add/remove) (玩家名称) (等级)` 来给居民一个等级或移除它。

### 城镇公告板

 

您可以为您的城镇设置自定义消息，称为城镇公告板。使用`/t set board (消息)”`更改您城镇的**城镇公告板**。

 

要移除城镇公告板，输入 `/t set board none`。此命令还会移除城镇居民进入服务器时显示的消息。

 

###  关闭/开放城镇

 

所有城镇默认都是关闭的（**仅限邀请**）。玩家只有在被城镇中的居民邀请后才能加入关闭的城镇。如果城镇是开放的，任何玩家都可以使用 `/t join (城镇名称)`加入该城镇。如果城镇是开放的，在其城镇信息屏幕中用 “(Open)” 表示。

 

要在您的城镇开放或关闭之间切换，使用 `/t toggle open` 。

 

#### 私人/公开城镇

 

所有城镇**默认**都是<u>私人</u>的。私人城镇只允许其居民、国家成员和盟国成员传送到其城镇出生点。如果城镇是公开的，服务器上的任何玩家都可以传送到其城镇出生点。如果城镇是公开的，在其城镇信息屏幕中用 “(<u>Public</u>)” 表示，并显示其<u>主区块</u>的**区块坐标**。

 

要在您的城镇私人或公开之间切换，使用`/t toggle public`。

 

可以使用 `/t set spawncost (价格)` 设置**外部人员**传送到公开城镇的<u>城镇出生点</u>的**费用**。

 

城镇的居民、国家成员和盟国成员不支付此城镇出生点费用。出生点费用资金从传送玩家的余额<u>转移</u>到城镇银行。

 

在仅限 Towny 的情况下，玩家可以使用 `t spawn (城镇名称)` 传送到公开城镇的城镇出生点。

 

###  城镇税收

 

城镇税收是在**每个** Towny 日开始时从居民那里收取的金额。资金从居民（**不包括具有免税、助理、副市长和市长等级的居民**）的余额转移到**城镇银行**。居民可以通过输入 `/res tax` 查看他们每个 Towny 日支付的税收。

 

1) 居民税收：居民税收是对居民作为您城镇一部分的税收。使用 `/t set taxes (百分比/金额)` 设置。

 	收集的居民税收类型可以使用 `/t toggle taxpercent` 切换。

2) 居民地皮税收：居民地皮税收（或简称"**地皮税收**"）是对居民拥有的每个居民地皮的税收。使用 `/t set plottax (金额)` 设置。没有足够资金支付居民地皮税收的居民将失去其居民地皮的所有权（**取决于他们无法支付税收的地皮数量**），丢失的地皮将被出售。

3) 商店地皮税收：商店地皮税收是对居民拥有的每个商店地皮的税收。使用 `/t set shoptax (金额)` 设置。没有足够资金支付商店地皮税收的居民将失去其商店地皮的所有权（取决于他们无法支付税收的地皮数量），并且他们的商店招牌将被移除。

4) 大使馆地皮税收：大使馆地皮税收是对玩家在您城镇内拥有的每个大使馆地皮的额外税收。使用 `/t set embassytax (金额)` 设置。没有足够资金支付大使馆地皮税收的玩家将失去其地皮的所有权（取决于他们无法支付税收的地皮数量）。

居民地皮税收与商店地皮税收和大使馆地皮税收是**分开**的。大使馆地皮所有者向您的城镇支付居民地皮税收和大使馆地皮税收。商店地皮所有者向您的城镇支付居民地皮税收和商店地皮税收。

 

###  城镇和地皮切换

 

切换更改其应用到的地块的游戏规则。

 

要在城镇中的所有地块中启用或禁用切换类型（城镇切换），使用 `/t toggle (切换类型)`。要在单个地块中启用或禁用切换类型（地皮切换）——由居民拥有或不拥有——使用 `/plot toggle (切换类型)`。



切换类型：

- pvp - 启用/禁用 PvP

- fire - 启用/禁用火势蔓延

- explosion - 启用/禁用爆炸

- mobs - 启用/禁用怪物生成

 

您可以在城镇信息屏幕中查看城镇切换。您可以通过输入 `/plot perm` 查看单个城镇地皮的地皮切换。

 

###  居民切换

 

居民切换是您购买居民地皮后自动应用于您的居民地皮的地皮切换。您可以使用 `/res toggle (切换类型)` 更改您的居民切换。

 

###  地皮类型

 

您的城镇中可以有不同类型的地皮，具有各种用途。默认情况下，当您从荒野占领区块时，它成为默认地皮：一个城镇地皮，除了被城镇占领外没有特殊用途。

 

要在您城镇中设置无主城镇地皮的地皮类型，使用 `/plot set (地皮类型)`。

 

地皮类型：

- default - 默认地皮

- shop - 商店地皮

- embassy - 大使馆地皮

- aren - 竞技场地皮

- wilds - 荒野地皮

- farm - 农场地皮

- bank - 银行地皮

- *jail -* *监狱地皮*

- inn - 旅店地皮

 

要将地皮重置为**默认地皮**，使用 `/plot set reset`。

 

您可以通过输入 `/t plots`查看您城镇的地皮类型（以及其他信息，如城镇规模、城镇地皮的预计每日收入等）。

 

**本服务器不启用监狱设置，因此所有关于监狱的部分的说明均无效。**

 

###  转让市长职位

 

市长可以放弃其高位，并将市长等级的头衔和权力授予城镇的另一个居民，使用 `/t set mayor (新市长名称)`。

 

前市长在放弃市长职位后将没有任何城镇等级。

 

###  城镇删除

 

删除城镇意味着从服务器中移除城镇的名称、领地、居民、城镇银行中的资金以及与城镇相关的所有内容。

 

只有市长能够删除其城镇。作为市长，您可以使用 `/t delete` 然后 `/confirm` 删除您的城镇。

 

 

###  城镇破产

 

当城镇银行资金达到 0（由于维护费用/国家税收）时，它将进入破产模式。城镇银行余额可以**继续低于 0**，因为由于维护费用或国家税收从中扣除资金。破产模式期间城镇银行低于 0 的值称为其“**债务**“。债务限制或债务上限是破产模式下的城镇在被删除前拥有的最大可能债务，它由城镇拥有的领地数量计算得出。债务上限可以在城镇信息屏幕中看到。

在破产模式期间，城镇面临一些**限制**。

要退出破产模式，必须在城镇银行中存入足够的资金以消除债务（`/t deposit`）。当城镇银行超过 0 时，城镇将与其所有居民一起恢复。然而，如果债务达到最大值（**这取决于城镇占领和购买的城镇地皮的总成本**），那么城镇将被服务器**删**除。

 

##  地皮信息

 

存在一些命令和功能，严格与非特定的单个领地相关。

 

使用 `/plot perm` 查看您所在区块的地皮信息屏幕。您还可以通过输入`/plot perm hud` 在您的记分板上显示地皮信息屏幕的**实时更新版本**。

 

地皮信息屏幕：

- 地皮类型

- 地皮所有者

- 地皮价格

- 地皮权限

- 地皮切换

 

荒野区块有一个单独的地皮信息屏幕，称为 Towny 世界屏幕，显示为荒野中的所有玩家应用的游戏规则和启用的权限、世界的战争设置以及与 Towny 相关的各种其他服务器设置。Towny 世界屏幕可以随时使用 `/tw` 查看。所有其他（被占领的）区块显示上述地皮信息屏幕。

 

#### 地皮组

 

地皮组允许居民一次轻松管理多个城镇地皮，例如权限和地皮销售。城镇工作人员可以创建地皮组并在其中包含任何城镇地皮，以及管理城镇内的任何地皮组。其他居民只能将自己的居民地皮包含在地皮组中，并管理仅包含其居民地皮的地皮组。

 

一个城镇中可以存在多个地皮组。您可以通过输入 `/t plotgrouplist` 查看城镇内所有地皮组的列表。

 

所有地皮组命令仅在您站在地皮组的任何地块内时才起作用（**例外是将地块添加到地皮组**）。

 

###  通用

 

要将您城镇的城镇地块包含在地皮组中，使用 `/plot group add (地皮组名称`)。如果地皮组不存在，它也将以您输入的名称创建。您也可以使用相同的命令将城镇地块从其当前地皮组转移到另一个地皮组。

 

可以通过在站在地皮组中的地块上时使用`/plot group rename (地皮组新名称)` 更改现有地皮组的名称。

 

要从地皮组中移除您所在的城镇地块，使用`/plot group remove`。要改为从地皮组中移除所有地块，请在地皮组的任何城镇地块上站立时输入 `/plot group delete`。此操作删除地皮组；移除地皮组中的最后一个地块也会删除地皮组。

 

###  管理

 

地皮组管理命令等同于地皮命令，并且以类似的方式工作。然而，地皮组管理命令适用于您所在的地皮组中的所有地块（与您仅所在的**单个城镇地块相对**）。

 

销售：使用`/plot group forsale (价格)` 将地皮组中的所有地块设置为出售。价格是地皮组中所有地块的价格，每个地块单独计算。使用 `/plot group notforsale`**撤回**地皮组的销售。

 

地皮类型：使用 `/plot group set (地皮类型)` 设置地皮组中所有地块的地皮类型，或使用 `/plot group set reset` 将它们全部**设置**为<u>默认地块</u>。

 

 当在同一个组中添加无主和居民地块时，请注意以下关于地皮类型的内容：

- 如果地皮组包含无主和居民地块，则使用`/plot group set`设置的地皮类型将**仅适用于无主地块**。

 :::tip

为了防止星号 解释的上述注意事项，强烈建议避免将无主城镇地块与居民地块分组在一起在同一个地皮组中。

 ~~~

-切换：使用 `/plot group toggle (切换类型)`为地皮组中的所有地块配置切换。

-权限：使用 `/plot group set perm (权限等级) (权限类型) (on/off)`为地皮组中的所有地块配置权限。
