# Towny 插件使用指南

Towny 是一个土地保护和城镇/国家模拟插件，允许玩家创建、保护和管理城镇，拥有土地，并组建国家和联盟。本指南全面描述 Towny 插件的各项功能。

- **第一部分（“城镇基础”）**：Towny 入门介绍，涵盖创建城镇的初步步骤、购买土地及基本功能。
- **第二部分（“权限”）**：土地保护机制及其配置方法。
- **其余部分**：城镇、国家相关功能及插件设置详解。

---

## 一、基本概念

### 区块与城镇日

- **区块（Chunk）**：世界中 16×16 的区域，垂直方向无限延伸。按 `F3 + G` 可显示区块边界。
- **荒野（Wilderness）**：未被占领的区块，默认允许所有玩家建造、破坏、使用容器，且 PvP 启用。
- **城镇日（Towny Day）**：等同于现实中的 1 天。Towny 按城镇日运作，每日开始时会在聊天栏提示。

### 城镇（Town）

- 由至少 1 名玩家（居民）组成的集合体，可将荒野区块圈为己用以保护。
- 被占领的区块称为**领地**或**城镇地皮（Plot）**：
  - **无主城镇地皮**：未被居民拥有的地皮。
  - **居民地皮**：被居民购买并拥有的地皮。

### 居民（Resident）

- 从属于某城镇的玩家。
- 首位创建城镇者为**市长（Mayor）**，拥有最高权限，可访问所有城镇地皮并执行全部命令。
- 市长可授予其他居民**城镇等级**以协助管理。

### 银行与税务

- **城镇银行**：城镇资金虚拟存储库。
  - 仅市长和副市长可取款，所有居民可存款。
  - 可通过 `/t bankhistory` 查看当日交易记录。
- **维护费**：每个城镇日服务器自动从银行扣除，用于维持城镇存在。
  - 余额不足 → 进入**破产状态**，银行开始负债。
  - 破产期间功能受限，居民可存款解除破产。
  - 最长维持 30 天，超期或负债达上限 → 城镇被删除。
- **其他税收**（默认为 0，由市长设置）：
  - **国家税**：加入国家的城镇需向国家缴纳。
  - **地皮税**：拥有地皮的居民向城镇缴纳。
  - **城镇税**：所有居民（除助手等）向城镇缴纳。
  - 无法支付 → 失去地皮、被移出城镇或国家。

### 国家（Nation）

- 由一个或多个城镇组成，其中一镇为**首都**，其市长即为**国家领导人**。
- 拥有独立银行，可对所属城镇征税。
- 可设置 `/nation spawn` 出生点，若为“公开”状态，非敌对玩家可传送。
- 可通过 `/nation sanctiontown` **制裁**城镇，阻止其加入国家或使用国家传送。

## 二、城镇基础

### 创建城镇与邀请玩家

```bash
/t new <城镇名称>      # 或 /t create <城镇名称>
```

- 创建需扣除资金（本服：5000 金币）。
- 名称仅支持英文、数字、下划线。
- 创建者自动成为市长，可随时用 `/t set name <新名称>` 更名。

**邀请与管理居民：**

```bash
/t add <玩家名>        # 邀请玩家
/accept <城镇名>       # 接受邀请
/t leave               # 居民退出（市长除外）
/t kick <玩家名>       # 市长踢出居民
```

**访问控制：**

```bash
/t set perm off        # 禁止所有居民访问公共区域（仅保留地皮权限）
```

### 领地占领

- 创建城镇时自动占领所在区块。
- **占领相邻区块**：
  ```bash
  /t claim             # 费用：1000 金币起，每块递增 10%，最高 150%
  ```
- **占领非相邻区块（前哨）**：
  ```bash
  /t claim outpost     # 费用：5000 金币；本服最多 4 个前哨
  ```
  - 每个分离的领地区域称为**城镇前哨**。
  - 首个占领区块为**前哨出生区块**，站立点为**前哨出生点**。
- **放弃区块**：
  ```bash
  /t unclaim           # 放弃当前区块；放弃前哨最后一块将删除整个前哨
  ```

**高级占领命令：**

```bash
/town claim <数字>         # 占领立方体区域（不推荐）
/town claim circle <数字>  # 占领圆形区域（不推荐）
/town claim fill           # 填充已围合区域内的所有未占领区块
/towny map                 # 查看附近区块信息
```

**领地数量限制：**

- 默认 100 区块，可购买最多 999 额外额度（100 金币/块）。
- 建立国家额外获得 1000 区块额度。

### 城镇维护费用

- 每日自动从银行扣除：`5 × 区块数` 金币/日，上限 500 金币/日。
- 区块越多，费用越高。规划时应避免无必要扩张。
- 居民可随时存款：
  ```bash
  /t deposit <金额>      # 在主区块或银行地皮上操作（国家内限制）
  ```

### 购买和拥有地皮

- 居民（含市长）可购买无主地皮作为**居民地皮**，享有独立建造与保护空间。
- 购买后地皮权限由居民控制，与其他居民隔离。
- 市长可撤回销售：
  ```bash
  /plot notforsale
  ```
- 居民放弃地皮：
  ```bash
  /plot unclaim          # 站在自己地皮上执行
  ```
:::tip
组织良好城镇应让居民地皮集中、相邻，便于管理。
:::
### 城镇出生点

- 创建时所在区块为主区块，位置为出生点，地面有粒子标识。
- 修改方法：
  ```bash
  /t set homeblock       # 设置新主区块
  /t set spawn           # 设置新出生点（基于当前位置）
  ```
- 建议将主区块设于城镇中心，方便居民快速抵达各区域。

### 加入国家

**益处：**
- 更多可占领区块
- 经济、影响力、社交等玩法增强

**税收：**
- 除维护费外，需支付国家税（由国家领导人设定）
- 可通过 `/towny prices` 查看

**加入流程：**
```bash
/nation invite <城镇名>   # 国家领导人邀请
/accept <国家名>          # 市长接受
/deny <国家名>            # 市长拒绝
```

## 三、权限系统

### 权限类型

| 类型 | 说明 |
|------|------|
| **Build（建造）** | 放置/破坏方块 |
| **Destroy（破坏）** | 破坏方块 |
| **Switch（切换）** | 操作门、拉杆、按钮等 |
| **ItemUse（物品使用）** | 使用容器、床、告示牌等 |

### 权限等级

**无主城镇地皮适用：**
- `M` Mayor（市长）
- `s` Assistant（助手）
- `r` Resident（居民）
- `a` Ally（盟国）
- `o` Outsider（外来者）

**居民地皮适用：**
- `f` Friend（地皮所有者）
- `M` Mayor
- `s` Assistant
- `r` Resident
- `a` Ally
- `o` Outsider
:::tip
 权限显示规则：允许时显示首字母，拒绝时显示 `-`。
:::
### 权限类别与配置

| 类别 | 适用对象 | 配置命令 |
|------|----------|----------|
| **一般城镇权限** | 所有无主地皮 | `/t set perm <等级> <类型> <on/off>` |
| **单个城镇权限** | 当前无主地皮 | `/plot set perm <等级> <类型> <on/off>` |
| **居民权限** | 所有自有地皮 | `/res set perm <等级> <类型> <on/off>` |
| **地皮权限** | 当前居民地皮 | `/plot set perm <等级> <类型> <on/off>` |

### Trust（信任）

为特定玩家授予地皮全部权限（建造、破坏、切换、物品使用）：

```bash
# 城镇级信任（所有地皮）
/t trust add <玩家名>
/t trust remove <玩家名>

# 地皮级信任（当前地皮）
/plot trust add <玩家名>
/plot trust remove <玩家名>
```
:::tip
移除信任后，玩家将受常规权限规则约束。
:::

### 玩家权限（Player-Specific Permissions）

为特定玩家单独配置权限：

```bash
/plot perm add <玩家名>      # 添加玩家
/plot perm remove <玩家名>   # 移除玩家
/plot perm gui               # 图形界面配置（点击头颅调整 4 类权限）
```

- 保存后生效；删除玩家将恢复默认权限规则。

### 重置权限

```bash
/t set perm reset            # 重置所有无主地皮为一般城镇权限
/res set perm reset          # 重置所有自有地皮为居民权限
/plot set perm reset         # 重置当前地皮为默认权限（根据地皮类型）
```
:::wraing
 重置不影响 Trust 和玩家权限设置。
:::

## 四、城镇信息与管理

### 城镇信息

```bash
/t                       # 查看自己城镇
/t <城镇名>              # 查看其他城镇
```

**显示内容：**
- 城镇名称、领地数、最大领地数、居民数（在线/总数）
- 市长、城镇等级、类型（开放/关闭，公开/私人）
- 银行余额、税收类型与金额、维护费用
- （国家内）国家名称与等级

### 城镇等级

| 等级 | 权限 |
|------|------|
| **Mayor（市长）** | 免税、全部指令权限 |
| **Assistant（助手）** | 免税（仅城镇）、认领区块、邀请玩家、改地皮权限、设公开状态、分配等级 |
| **Sponsor（赞助者）** | 免税、认领区块、改地皮权限、邀请玩家 |
| **VIP** | 免税 |

```bash
/t rank add <玩家名> <等级>
/t rank remove <玩家名> <等级>
```

### 城镇公告板

```bash
/t set board <消息>      # 设置公告
/t set board none        # 移除公告（同时移除登录提示）
```

### 开放/关闭 城镇（Open/Closed）

- **关闭（默认）**：仅限邀请加入。
- **开放**：任何玩家可用 `/t join <城镇名>` 加入。
- 切换命令：
  ```bash
  /t toggle open
  ```

### 私人/公开 城镇（Private/Public）

- **私人（默认）**：仅居民、国家成员、盟国可传送至出生点。
- **公开**：任何玩家可传送（需付费）。
- 切换命令：
  ```bash
  /t toggle public
  /t set spawncost <价格>  # 设置外部玩家传送费用
  /t spawn <城镇名>        # 传送至公开城镇出生点（居民/盟友免费）
  ```
:::tip
🔍 区分概念：**Open/Closed** 控制加入方式，**Public/Private** 控制传送权限。
:::
### 城镇税收

每日自动从居民余额扣款至城镇银行（免税等级除外）：

| 税种 | 命令 | 说明 |
|------|------|------|
| **居民税** | `/t set taxes <金额/百分比>`<br>`/t toggle taxpercent` | 对居民征收 |
| **地皮税** | `/t set plottax <金额>` | 每块居民地皮；欠税 → 失去地皮 |
| **商店税** | `/t set shoptax <金额>` | 每块商店地皮；欠税 → 失去地皮+移除招牌 |
| **大使馆税** | `/t set embassytax <金额>` | 每块大使馆地皮；与地皮税叠加 |

居民可查看税额：`/res tax`

### 城镇与地皮切换（Toggles）

**城镇级切换（影响所有地皮）：**
```bash
/t toggle pvp          # 启用/禁用 PvP
/t toggle fire         # 火势蔓延
/t toggle explosion    # 爆炸
/t toggle mobs         # 怪物生成
```

**地皮级切换（当前地皮）：**
```bash
/plot toggle <类型>
```

查看切换状态：
- 城镇信息屏
- `/plot perm`（单个地皮）

### 居民切换（Resident Toggles）

购买地皮后自动应用，可自定义：

```bash
/res toggle bedspawn           # 床重生
/res toggle boardertitles      # 跨界显示城镇名
/res toggle ignoreotherchannels # 忽略其他城镇聊天
/res toggle ignoreplots        # 忽略地皮提示
/res toggle map                # 进出区块显示地图
/res toggle plotgroup          # 自动加入地皮组
/res toggle townborder         # 进入城镇显示边界粒子
/res toggle constantplotborder # 持续显示地皮边界
/res toggle plotborder         # 进入地皮短暂显示边界
/res toggle townclaim          # 走入自动认领区块（再输一次关闭）
/res toggle townunclaim        # 走入自动放弃区块（再输一次关闭）
```

## 五、地皮类型

默认地皮无特殊功能。设置命令：

```bash
/plot set <类型>
/plot set reset    # 重置为默认
/t plots           # 查看城镇地皮类型与收入预估
```

### 各类型说明

| 类型 | 命令 | 特性 |
|------|------|------|
| **默认** | `/plot set default` | 普通地皮，可直接 `/plot forsale` 出售 |
| **商店** | `/plot set shop` | 可设额外商店税；市长可设默认售价 `/town set shopprice` |
| **竞技场** | `/plot set arena` | 强制开启 PvP 与友伤；禁用生命回复；物品无耐久损耗 |
| **大使馆** | `/plot set embassy` | 可设额外大使馆税；市长可随时收回；离镇不失去所有权；可转为商店 |
| **旷野** | `/plot set wilds` | 允许破坏特定方块（矿石、树木等）；查看列表：`/towny wildsblocks` |
| **旅店** | `/plot set inn` | 任何人可用床设重生点（敌人除外） |
| **监狱** | `/plot set jail` | ⚠️ 本服已禁用监禁功能 |
| **农场** | `/plot set farm` | 仅允许种植/破坏农作物方块；可杀动物；查看列表：`/towny allowedblocks` |
| **银行** | `/plot set bank` | 限制银行存取款；本服无实际作用 |

:::tip
 本服规则：城镇区块内死亡**不掉落物品与经验**。
:::

## 六、其他功能

### 转让市长职位

```bash
/t set mayor <新市长名>
```

- 前市长失去所有等级。

### 城镇删除

```bash
/t delete
/confirm
```

- 仅市长可执行，将移除城镇所有数据。

### 城镇破产

- 银行余额 ≤ 0 → 进入破产，产生债务。
- 债务上限 = 领地总成本，超限 → 城镇删除。
- 解除破产：存款使余额 > 0（`/t deposit`）。

### 地皮信息

```bash
/plot perm          # 查看当前地皮信息
/plot perm hud      # HUD 实时显示
/tw                 # 荒野世界规则（Towny World）
```

**显示内容：** 地皮类型、所有者、价格、权限、切换状态。

### 地皮组（Plot Groups）

将多个地皮归组统一管理（权限、销售、类型等）。

**查看与创建：**
```bash
/t plotgrouplist                # 列出所有地皮组
/plot group add <组名>          # 添加当前地皮至组（不存在则创建）
```

**管理命令（需站在组内地皮）：**

| 操作 | 命令 |
|------|------|
| 重命名 | `/plot group rename <新名>` |
| 移除单块 | `/plot group remove` |
| 删除整组 | `/plot group delete` |
| 批量出售 | `/plot group forsale <总价>` |
| 撤回销售 | `/plot group notforsale` |
| 设置类型 | `/plot group set <类型>` 或 `reset` |
| 切换设置 | `/plot group toggle <类型>` |
| 权限设置 | `/plot group set perm <等级> <类型> <on/off>` |
:::waring
 混合无主与居民地皮时，`/plot group set` 仅影响无主地皮，建议分组管理。
:::

## 七、服务器特定配置

| 项目 | 数值 |
|------|------|
| 创建城镇 | 5000 金币 |
| 创建国家 | 50000 金币 |
| 认领相邻区块 | 1000 金币起，每块 +10%，最高 150% |
| 认领前哨 | 5000 金币/个，最多 4 个 |
| 购买区块额度 | 100 金币/块（最多 +999） |
| 城镇日维护费 | `5 × 区块数` 金币/日，上限 500 |
| 国家日维护费 | `100 × 城镇数` 金币/日 |
